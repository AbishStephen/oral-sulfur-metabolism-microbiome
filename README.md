# Sulfur metabolism and immune-microbial networks across oral niches in periodontal disease

This repository contains processed datasets and analysis scripts used in the study:

Stephen A et al. Sulfur metabolism and immune-microbial networks across oral niches in periodontal disease.

## Data files

clinical_data.csv  
Clinical measurements including plaque index, bleeding on probing, diagnosis groups, and volatile sulfur compound measurements (H2S and CH3SH).

gcf_cytokines.csv  
Matrix of gingival crevicular fluid cytokine concentrations.

microbial_data.csv  
Species-level microbial feature table derived from 16S rRNA gene sequencing.

sampleinfo.csv  
Mapping file linking sample IDs to patient IDs and oral niches (interdental, subgingival, tongue).

qpcr_data.csv  
qPCR abundance measurements for selected bacterial taxa.

path_abun_unstrat.tsv.gz  
PICRUSt2 predicted enzyme abundance table.

## Scripts

R scripts used to reproduce all analyses and figures in the manuscript.
